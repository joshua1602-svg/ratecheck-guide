Methodology

Data Source

All comparable data is drawn from the Valuation Office Agency's published bulk data, which contains the rateable values and property characteristics used to compile the 2026 rating list. This is the same dataset the VOA uses internally when assessing rateable values.

Selection Criteria

Comparable properties were selected based on geographic proximity, matching business type classification, and similar net internal area. The selection algorithm identifies properties within the same billing authority area first, then expands the search radius if insufficient comparables are found locally.

Valuation Approach

The estimated rateable value is derived using a weighted average of comparable rates per square metre, applied to the subject property's floor area. Weights are assigned based on similarity across multiple dimensions including size, location, and — where layout data is provided — structural configuration.

{% if layout_adjustment_applied %}

Layout Weighting

The ratepayer provided detailed layout information for the subject property. This data was used to calculate a structural fingerprint comprising storage-to-trading ratios, floor configuration, and kitchen positioning. Each comparable was scored against this fingerprint, and comparables with higher structural similarity received greater weight in the valuation model. This approach ensures the estimated rateable value more closely reflects the economic characteristics of the subject property rather than relying solely on headline floor area.

{% endif %}

Antecedent Valuation Date

All values reflect market conditions as of 1 April 2024, the antecedent valuation date prescribed by the VOA for the 2026 rating list. This is the reference date against which all rateable values in the current list are assessed.

Limitations

This analysis is based on publicly available data and information provided by the ratepayer. It does not include a physical inspection of the subject property or any of the comparable properties. The estimated rateable value is indicative and intended to support a Check Challenge Appeal submission. It does not constitute a formal valuation by a qualified surveyor.
